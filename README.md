# Independence Day Celebration Web App:
A creative and interactive web project built to celebrate **Independence Day of India**.
This project combines patriotism with web development by including animations, quiz interaction, and certificate generation.

# 🎯 Features:
✨ **Animated UI**
* Flowing Indian flag animation 🇮🇳
* Falling flower/emoji effects 🌸

🧠 **Independence Day Quiz**:
* Multiple choice questions
* User participation feedback

🎓 **Certificate Generator**:
* Enter your name..
* Generate personalized certificate
* Download as image 📄

🎨 **Modern Design**
* Responsive layout 📱
* Gradient + background image
* Clean and user-friendly UI
* 
## 🛠️ Tech Stack
* HTML5
* CSS3
* JavaScript
* html2canvas (for certificate download)
